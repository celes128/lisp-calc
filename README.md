# lisp-calc
A Python program that evaluates arithmetic expressions written in a lisp-like syntax.

# Examples
(+ 1 2) evaluates to 1 + 2 = 3.
(- 6 20) evaluates to 6 - 20 = -14.

# Nested expressions
(+ 1 (* 2 3)) evaluates to 1 + 2*3 = 7.
(+ 1 (* 2 3) (/ 100 50)) evaluates to 1 + 2*3 + 100/50 = 9.

Supported operators: + - * /
